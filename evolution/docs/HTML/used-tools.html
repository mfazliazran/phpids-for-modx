Used tools and libraries

All tools and libraries, that are mentioned, are part of this package.

PHPIDS

The package is based on PHPIDS, available at http://phpids.org/. PHPIDS is a PHP intrusion detection system. The inspiration for this package is based on an article by Daniel Bachfeld, published on 1th. May 2009 at http://www.h-online.com/security/features/Getting-started-with-the-PHPIDS-intrusion-detection-system-746233.html.

This article is also a god start for understanding PHPIDS.

PHPIDS is part of this package since version 0.6.4. The version number of this plugin refers to the version number of PHPIDS. This means, that 0.6.5 is the current version of PHPIDS in this packet.

jQuery

The module rendering is based an jQuery, which is available at www.jquery.com. The current version used in this package is 1.5.1.

jQueryUI

The jQuery package is needed for the layout and some functionality of the module. The current version used in this package is 1.8.10. jQueryUI is available at http://jqueryui.com/.

jqGrid

The grid component used for listings is the jQuery and jQueryUI based jqGrid. The current version used in this package is 3.8.1. You may find more information about jqGrid at http://www.trirand.com/blog/.

jQuery blockUI plugin

The blockUI plugin is used, while AJAX functions are called. The current version used in this package is 2.37. This jQuery plugin is available at http://malsup.com/jquery/block/.
